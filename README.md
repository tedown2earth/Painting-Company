# Perfect Touch Painters Website.

## Project overview
The Perfect Touch Painters Website is a front-end web application designed to represent a professional painting company. The website allows customers to learn about the company, view its services, and submit painting service requests. All submitted requests are stored locally and displayed on the requests page.

## Features
 1. Home Page -> Introduces the company.

 2. About Page -> Displays the company's philosophy, mission, vision and detailed services.

 3. Contact Page -> Allows users to submit painting service requests.

 4. Requests Page -> Displays all submitted requests.

 5. Local storage -> Requests are saved using the browser's local storage.

## How the Website Works
 
 1. A user submits a painting request through the contact page

 2. JavaScript validates the form inputs

 3. The requests are saved in the browser's localStorage

 4. The requests page retrieves stored requests and display's them.

 5. Requests can be deleted, which updates both the interface and localStorage.

 ## How to Run the project

 1. Open index.html 
 
 2. Navigate through the site using the navbar

 3. Submit requests through the contact page

 4. View submitted requests on the requests page
